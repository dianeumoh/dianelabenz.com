Dashboard Iterations

How will the home dashboard address user goals?

After completing the initial application process, the user is taken to the home page via a welcome email. For Lantern's home page dashboard, I thought about the following questions:

  • How will the user know what to do next?
  • How do I reduce the amount of steps and time to search and apply for jobs?
  • How do I encourage the user to return to the Lantern platform?

Dashboard V1: Utilizing a progress bar and checklist to show onboarding steps

In early iterations, I focused on a progress bar and checklist to encourage developers to complete the onboarding process, shortening the time it would take to apply for available jobs.

I incorporated the job statistics card into the homepage dashboard to communicate the user's progress and reinforce the value of the Lantern platform.

Dashboard V2: Reducing the number of clicks to see available jobs

I spent time iterating on how the dashboard would evolve with the developers on their job hunt from initial login to three months on the platform. It became clear that the reason a developer would spend time on the platform is to find a job. While job statistics were helpful to the user, it wasn't relevant enough to the main user goal of quickly finding and applying for jobs.

The homepage dashboard evolved to the jobs dashboard giving the user quicker access to jobs on the Lantern platform from login.

Dashboard V3 (final): Creating the best experience for users to find the latest jobs

The jobs dashboard utilized card components to house content such as:

  • New For You: Onboarding instructions or platform messages
  • Jobs You Might Like: Personalized job recommendations based on the user's profile and activity
  • My Profile: At a glance profile and skill certification information

Unlike other job boards, Lantern wanted to place less emphasis on the company and more emphasis on the technical stack and job description. Based on competitive and user research, I incorporated tech stack language logos to aid scannability in lieu of company logos.
